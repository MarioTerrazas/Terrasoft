# Comandos de Git

---

## Ver estado

```bash
git status
```

---

## Agregar archivos

```bash
git add .
```

---

## Crear commit

```bash
git commit -m "Mensaje"
```

---

## Subir cambios

```bash
git push
```

---

## Descargar cambios

```bash
git pull
```

---

## Ver ramas

```bash
git branch
```

---

## Ver repositorio remoto

```bash
git remote -v
```

---

## Historial

```bash
git log --oneline
```

---

## GitHub CLI

### Estado

```bash
gh auth status
```

### Crear repositorio

```bash
gh repo create
```

### Clonar

```bash
gh repo clone
```
